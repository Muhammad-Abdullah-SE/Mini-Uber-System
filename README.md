# Mini-Uber-System
A robust, file-based ride-sharing management system implemented in C++. This project demonstrates the application of core Data Structures and Algorithms (DSA), Object-Oriented Programming (OOP), and efficient file handling to manage riders, drivers, and ride lifecycles.✨ Key FeaturesMulti-Role Access: Dedicated portals for Admins, Drivers, and Riders, each with secure login authentication and password recovery.Ride Lifecycle Management: * Riders can book rides (calculating fare based on distance), cancel pending requests, and rate drivers upon completion.Admins manage the queue of pending requests and assign them to available drivers.Drivers maintain a profile tracking their total earnings, average rating, and trip history.Persistent Storage: Automatically loads and saves data (Drivers, Riders, Completed/Cancelled Rides) to .csv files, ensuring data is never lost between sessions.Advanced Analytics & Reporting: * Financial summaries (Total earnings, average trip distance).Performance tracking (Top-rated drivers, most active riders).Real-time statistics on pending vs. completed rides.🛠️ Technical HighlightsData Structures: * std::vector for dynamic user management.std::queue for managing ride requests in a First-In-First-Out (FIFO) manner.struct models for clean data organization.Algorithms:Binary Search: Implemented for $O(\log n)$ efficiency when looking up users by ID.Bubble Sort: Used to organize driver lists by rating (highest first) or alphabetically.Lambda Expressions: Utilized for custom sorting logic.Input Validation: Robust error-handling functions ensure that names contain only letters, contacts contain only digits, and ratings stay within the 1–5 range.Time Integration: Uses the <ctime> library to timestamp the start and end of every ride accurately.📂 File StructureThe system maintains four database files to ensure modularity:drivers.csv: Driver credentials, status, ratings, and earnings.riders.csv: Rider credentials and booking history.completed_rides.csv: Historical log of all successful trips.cancelled_rides.csv: Records of rides cancelled by riders.💻 How to RunClone the repository.Compile using any C++ compiler (e.g., g++ main.cpp -o MiniUber).Run the executable.Use the default
